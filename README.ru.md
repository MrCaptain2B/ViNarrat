# ViNarrat

Интерактивный режиссёр сцен для **Foundry Virtual Tabletop v13/v14**.  
Ведущий управляет сценой (фон, портреты, диалог), игроки видят всё в реальном времени и управляют своими персонажами.

> **Версия 0.7.0** — Активная разработка

---

## Возможности

### Инструменты ведущего
- **Полноэкранная сцена** — окно диалога, индикатор говорящего, портреты, фон
- **Управление портретами** — добавление, перетаскивание, масштаб, отражение, блокировка; импорт из листов актёров
- **Фоны-локации** — фоновые изображения с регулировкой яркости, группировка и теги
- **Управление сценой** — диалог (один или два столбца), выбор говорящего, темы (цвета фона и акцента)
- **Трансляция** — отправка сцены всем онлайн-игрокам (или только на сцене) в реальном времени
- **Приглашение** — адресное приглашение конкретного игрока
- **Сохранения сцены (Presets)** — снепшоты сцены: фон, портреты, говорящий, настройки диалога

### Система сценариев (Script Engine)
- **Сценарии** — многошаговые автоматические последовательности
- **Типы шагов**: Scene (полное состояние сцены), Pause (пауза N секунд), Transition (эффекты перехода)
- **Воспроизведение** — play, пауза, шаг вперёд/назад, печатающийся текст, анимированные переходы
- **Редактор** — перетаскивание шагов, редактирование полей, экспорт/импорт JSON
- **Открывающий переход** — затемнение с чёрного; **закрывающий переход** — уход в чёрный

### Триггеры регионов
- Привяжите любой сценарий к **Региону** Foundry
- Автоматический запуск при входе **подконтрольного игроку токена**
- Использует стандартные инструменты Регионов

### Возможности игроков
- **Просмотр в реальном времени** — синхронизация с трансляцией ведущего
- **Мой портрет** — панель своего портрета: имя, титул, эмоции, изображения
- **Полоска эмоций** — клик меняет выражение портрета (видно всем)
- **Attention / Claim** — ✋ на портрете → пульсация у ведущего → ✓ подтверждение
- **Переподключение** — `/vnrejoin` сбросит состояние после обновления страницы
- **Сохранение сессии** — состояние переживает обновление страницы

### Права доступа
Для каждого действия настраивается минимальная роль (Player / Trusted / Assistant / GM):

| Право | По умолчанию | Что даёт |
|-------|-------------|----------|
| `permManage` | GM | Полное управление сценой |
| `permBroadcast` | GM | Запуск/остановка трансляции |
| `permApproveClaims` | GM | Подтверждение Attention от игроков |
| `permAddRequests` | Player | Отправка запросов ведущему |

## Быстрый старт

```
/vn           Открыть VN-оверлей (ведущий)
/vnreq текст  Отправить запрос ведущему
/vnportrait   Открыть панель своего портрета (игрок)
/vnedit       То же, что /vnportrait
/vnrejoin     Пересинхронизация после разрыва
```

### Первый запуск (ведущий)
1. `/vn` — открыть оверлей
2. Панель **Portraits** → **Import from Actors** → выбрать актёров
3. Панель **Locations** → добавить фоны
4. **Scene Control** → ввести диалог, выбрать говорящего
5. Нажать **Broadcast** — игроки увидят сцену
6. **Create Script** — создать сценарий для автоматической смены сцен

## Система сценариев

### Сценарии vs Сохранения (Presets)

| Сохранения (Presets) | Сценарии (Scripts) |
|----------------------|--------------------|
| Один снепшот сцены | Последовательность шагов |
| Фон + портреты + говорящий | Диалог, эмоции, паузы, переходы |
| Ручная загрузка | Автовоспроизведение |
| Нет логики | Пошаговое выполнение |

### Типы шагов

| Тип | Эффект |
|-----|--------|
| **Scene** | Применяет полное состояние — фон, портреты, диалог, говорящий |
| **Pause** | Ожидание N секунд, без изменений сцены |
| **Transition** | Визуальный эффект (fadeToBlack или crossfade) — не меняет состояние |

### Управление воспроизведением
- **Клик в пустоту** — следующий шаг (первый клик сбрасывает печатание, второй — переход)
- **Prev / Next / Stop** — навигация и выход
- **Play / Pause** — автоматическое воспроизведение
- Портреты заблокированы, панель говорящего скрыта, сцена неинтерактивна

### Редактор
- **Load step** — загрузить состояние шага в редактор сцены
- **Edit state** — изменить сцену, затем **Save** для обновления шага
- **Up / Down** — перестановка шагов
- **Delete** — удалить шаг
- **Add** — Scene, Pause или Transition (с выбором типа и длительности)
- Шаблон по умолчанию: Opening Transition → Scene → Closing Transition

## Триггеры регионов

1. Создайте **Регион** на сцене
2. Добавьте поведение → **ViNarrat Script**
3. Нажмите **Select Script** → выберите сценарий
4. Когда **подконтрольный игроку токен** войдёт в регион, сценарий запустится **один раз**
5. Повторный вход не запустит сценарий заново

## API

```js
game.freevisualnovel.open()                // Открыть VN-оверлей
game.freevisualnovel.close()               // Закрыть VN-оверлей
game.freevisualnovel.setBackground(path)
game.freevisualnovel.addPortrait(data)
game.freevisualnovel.addRequest(text)
game.freevisualnovel.clearStage()
game.freevisualnovel.importActorPortraits()
```

## Установка

1. Foundry VTT → **Add-on Modules** → **Install Module**
2. Вставить URL манифеста:
   ```
   https://raw.githubusercontent.com/MrCaptain2B/FreeVisualNovel/master/module.json
   ```
3. Включить модуль в мире

Или склонировать для разработки:
```bash
git clone https://github.com/MrCaptain2B/FreeVisualNovel.git
```
